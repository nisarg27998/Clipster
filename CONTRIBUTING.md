### Reporting Bugs
* Check the [Issues tab](https://github.com/nisarg27998/Clipster/issues) to see if the bug has already been reported.
* If not, open a new issue. Include steps to reproduce the bug and details about your OS.

### Suggesting Enhancements
* Open an issue with the tag `enhancement`.
* Describe the feature you'd like to see and why it would be useful.

### Pull Requests (Code Contributions)
1. **Fork** the repository.
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/Clipster.git`
3. **Create a branch** for your fix or feature: `git checkout -b feature/your-feature-name`
4. **Install dependencies**: `pip install -r requirements.txt` (or manually install `customtkinter`, `Pillow`, and `yt-dlp`).
5. **Make your changes** and test them!
6. **Commit** your changes: `git commit -m "Add some feature"`
7. **Push** to the branch: `git push origin feature/your-feature-name`
8. **Open a Pull Request** against the `main` branch.
